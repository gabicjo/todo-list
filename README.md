# O que é isso?
Esse é um pequeno site que funciona como uma to-do list, com funções como adicionar tarefas, marcar e desmarcar tarefas e excluir tarefas concluidas. E é só isso mesmo, o objetivo dele é ser simples e direto, sem complicações.
# A identidade visual:
Esse site tem como base a paleta de cores do site https://pierre.finance/, utilizando o verde neon, preto e branco como base.
